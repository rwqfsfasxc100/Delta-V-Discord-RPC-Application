# Delta-V-Discord-RPC-Application
Docker application that acts as a pipeline between Delta-V and Discord. Needs the client mod to function.

> [!WARNING]
> Requires the client mod to operate.
> See [it's repository page for more information](https://github.com/rwqfsfasxc100/Delta-V-Discord-RPC)
>
> Due to limitations with the library used for Discord RPC, and a lack of hardware for testing on my own end, no OS-X support will be provided of current.
